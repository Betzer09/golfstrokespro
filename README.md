#GolfTrack Pro
Helps you track golf swings while on the course

## Features 
* You can click on the location icon to track you club swing distance
* Add a club to every swing
* Count your strokes while on the course

## Technoligy 
* CoreLocation
* SwiftData
* SwiftUI
